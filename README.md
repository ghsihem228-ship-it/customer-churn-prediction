# Customer Churn Prediction

## Overview
Machine Learning project to predict customer churn using real-world telecom customer data.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Logistic Regression

## Project Steps
- Loaded and explored customer churn dataset
- Cleaned missing and invalid values
- Converted categorical variables into numerical format
- Trained a Logistic Regression model
- Evaluated model performance using accuracy and classification report

## Goal
Predict whether a customer is likely to leave the company based on customer behavior and service features.
